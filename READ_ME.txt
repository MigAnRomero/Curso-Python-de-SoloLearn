Los siguiente archivos guardados y subidos al repositorio de GitHub son para fines educativos propios del
autor mientras sigue aprendiendo el uso y manejo de usar el lenguaje de programación de alto nivel,
Python a través del curso virtual ofrecido por SoloLearn.

Además, la carpeta con el nombre "Jugando con Python", son códigos sacado del libro "Curso: Python para
principiantes" de Eugenia Bahit (2012), un libro distribuido bajo la licencia
